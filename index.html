Projects

Lambda School - User Interface Sprint Challenge

This project explores User Interface and Git and showcases Semantic HTML, CSS Fundamentals, CSS Flexbox Module, and Git. I created a multi page website, adding to the missing HTML elements as well as CSS specificity problems that needed to be solved. I also created an additional web page that was linked to from the navigation (which you can see if you hover your mouse over the image).

Lambda School - Responsive Web Design I

This project expands our CSS skills into multiple devices. I created an adaptive version of the "Great Idea" design and worked through design mock ups for desktop, tablet, and mobile devices, by using max-width media queries at 800px and 500px.
